# The Luanti source repositories have moved to the new [luanti-org](https://github.com/luanti-org/) organisation!

Old links pointing to specific repositories, e.g. https://github.com/minetest/minetest and subsequent subpages, will redirect to the new location under https://github.com/luanti-org/luanti. However the organisation page itself does not redirect as it needs to remain in order to prevent someone from taking the name and breaking repository redirects.

Otherwise what remains in this organisation are obsolete repositories that are either archived or otherwise no longer useful going forward.
